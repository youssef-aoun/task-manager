GET /api/v1/projects
List all projects

Returns a list of projects the user owns or has joined.

Examples

[
  {
    "id": 1,
    "name": "My Project",
    "description": "An awesome project",
    "owner_id": 5
  }
]

Params

Param name Description
type
optional

Filter by type. **Must be one of: ‘owned’ (projects owned by user) or ‘joined’ (projects user has joined). Default: ‘owned’.**

Validations:

  • Must be a String

Code: 200

Successful response
